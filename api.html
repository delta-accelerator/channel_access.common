channel_access.common.datetime_to_epics(timestamp)¶

Convert a datetime object to an epics timestamp tuple.

Parameters:

timestamp (datetime) – A datetime object. Is this is a naive datetime object the timezone is assumed to be UTC.

Returns:

A (seconds, nanoseconds) tuple with an origin of EPICS_EPOCH in timezone UTC.

channel_access.common.epics_to_datetime(timestamp)¶

Convert an epics timestamp tuple to a datetime object.

Parameters:

timestamp – A (seconds, nanoseconds) tuple with an origin of EPICS_EPOCH in timezone UTC.

Returns:

An aware datetime object with UTC timezone.
